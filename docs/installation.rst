Installation
============

We've made this so easy that even your younger brother could do it (the one you don't talk about).

1. **Download the jar:** Grab the latest release. It's the file ending in ``.jar``. No, not the source code.
2. **Drop it in:** Throw that jar into your server's ``plugins/`` folder. Treat it with respect, but don't be afraid to be firm.
3. **Restart your server:** Or use a plugin loader if you like living on the edge.
4. **Profit:** That's it. You're done. Go have a snack.

**Requirements:**
- A Minecraft server (obviously).
- LuckPerms (because, you know, the name).
- Java 21 (we like to stay modern).
- A positive attitude (optional but recommended).
